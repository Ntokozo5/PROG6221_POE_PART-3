## Project Overview
This is a WPF C# cybersecurity chatbot that educates users about online safety while allowing them to manage cybersecurity tasks. The application stores tasks in a MySQL database and includes an interactive cybersecurity quiz.

## Features
- Conversational chatbot with over 60 cybersecurity responses.
- Voice greeting when the application starts.
- Cybersecurity awareness quiz.
- Create, view and manage cybersecurity tasks.
- MySQL database integration for persistent task storage.
- Activity log to track user actions.
- Friendly graphical user interface built with WPF.

## Technologies Used
- C#
- WPF (.NET)
- MySQL
- Visual Studio

## Requirements
- Visual Studio 2022 or later
- .NET Desktop Development workload
- MySQL Server
- MySQL Connector for .NET

## How to Run
1. Open the solution in Visual Studio.
2. Ensure MySQL Server is running.
3. Update the database connection string in `MainWindow.xaml.cs` if necessary.
4. Build and run the project.

## Project Structure
- `MainWindow.xaml` – User interface.
- `MainWindow.xaml.cs` – Main chatbot logic.
- `TaskRepository.cs` – Database operations.
- `CyberTask.cs` – Task model.
- `QuizQuestion.cs` – Quiz model.
- `ActivityEntry.cs` – Activity log model.
- `Database` – Database scripts/files.
